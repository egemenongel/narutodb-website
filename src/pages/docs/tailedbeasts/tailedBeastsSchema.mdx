---
title: Tailed Beast Schema
---

| Key            | Type        | Description                       |
| -------------- | ----------- | --------------------------------- |
| `id`           | String      | id of the character               |
| `name`         | String      | name of the character             |
| `images`       | String(URL) | url of character image            |
| `debut`        | Objects     | debut made by character           |
| `personal`     | Objects     | Personal details of the character |
| `family`       | Objects     | Family details of the character   |
| `jutsu`        | Array       | Jutsu of the character            |
| `natureType`   | Array       | Nature Type of the character      |
| `uniqueTraits` | Array       | Unique Traits of the character    |
