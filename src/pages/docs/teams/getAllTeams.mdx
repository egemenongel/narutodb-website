title: Get all Teams
---

```js
GET https://api.narutodb.xyz/team
```

```js
curl -X 'GET' \
        'https://api.narutodb.xyz/team' \
     -H 'accept: application/json'
```

#### Pagination and Limit

```js
https://api.narutodb.xyz/team?page=2&limit=5
```

```js
curl -X 'GET' \
  'https://api.narutodb.xyz/team?page=2&limit=5' \
  -H 'accept: application/json'
```
