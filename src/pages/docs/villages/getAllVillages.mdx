title: Get all Villages
---

```js
GET https://narutodb.xyz/api/village
```

```js
curl -X 'GET' \
        'https://narutodb.xyz/api/village' \
     -H 'accept: application/json'
```

#### Pagination and Limit

```js
https://narutodb.xyz/api/village?page=2&limit=12
```

```js
curl -X 'GET' \
  'https://narutodb.xyz/api/village?page=2&limit=12' \
  -H 'accept: application/json'
```
