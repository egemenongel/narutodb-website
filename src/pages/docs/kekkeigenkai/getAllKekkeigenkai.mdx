```js
GET https://api.narutodb.xyz/kekkei-genkai
```

```js
curl -X 'GET' \
        'https://api.narutodb.xyz/kekkei-genkai' \
     -H 'accept: application/json'
```

#### Pagination and Limit

```js
https://api.narutodb.xyz/kekkei-genkai?page=1&limit=15
```

```js
curl -X 'GET' \
  'https://api.narutodb.xyz/kekkei-genkai?page=1&limit=15' \
  -H 'accept: application/json'
```
